Use Cases

마케터

분석용 대시보드

마케터는 개인용 대시보드에서 자신이 맡은 캠페인이나 여러 광고주의 성과를 자유롭게 분석합니다. 필요한 지표를 위젯으로 조합해 매일 반복되는 확인 과정을 단순화하고, 협업 대시보드를 통해 동료와 함께 광고 성과를 관리할 수도 있습니다.

마케터 & 광고주

광고주 리포트

실무 마케터는 광고주용 대시보드를 만들어 웹 리포트로 공유합니다. 광고주는 웹·모바일 환경 어디서든, 별도 파일 없이 최신 데이터가 자동 반영된 리포트를 확인할 수 있습니다.

보고 업무 자동화 + 광고주의 리포트 접근성 개선
조직 관리자

통합 관리 대시보드

관리자는 조직에서 관리하는 여러 광고주와 매체의 성과를 하나의 대시보드에서 통합 조회합니다. 성과가 좋은 광고주, 주의가 필요한 광고주를 빠르게 식별하고, 리소스와 전략을 효율적으로 배분할 수 있습니다.

대시보드 유형

맵핑과 대시보드 모두 개인용과 광고주용 두 가지 유형으로 제공됩니다. 개인용은 본인 계정에서 개인적으로 사용하는 공간으로 OWNER 권한 내 여러 광고주 데이터를 조합 가능하여 개인 맞춤 분석이나 통합 인사이트 확인에 유용합니다. 광고주용은 광고주 단위로 생성되는 공유 공간으로 해당 광고주 권한을 가진 사용자가 함께 접근할 수 있어 팀 단위 협업에 최적화되어 있습니다.
